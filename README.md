EventSync is a modern, responsive, full-stack event management system. It features real-time role-based workspaces for both Administrators (comprehensive CRUD, analytical statistics, feedback logs) and Regular Users (browsing, searching, event scheduling with meteorological integration). The platform redefines how organizers deploy experiences and how attendees secure their entry.

## 🚀 Key Features

* **🔐 User Authentication** - Secure login and registration system for admin/organizers and users.
* **🗓️ Event Creation & Management (CRUD)** - Create, update, view, and delete events easily.
* **🔎 Search & Filtering** - Search events by category, date, location, or keywords.
* **📊 Dashboard & Statistics** - View event analytics, total events, and bookings statistics.
* **📅 Calendar Integration** - Organized calendar view for upcoming and scheduled events.
* **🎟️ Booking & Registration System** - Users can register and book seats for events.
* **📥 Notifications** - Receive updates related to registrations, events, and announcements.
* **💰 Financial Tracking** - Tracks event revenue.
* **⭐ Feedback System** - Collect reviews and feedback from attendees.
* **🌤️ API Integration** - Integrates external APIs such as open-meteo weather APIs for event planning assistance.

## 🛠️ Technologies Used

* **Frontend:** React.js (Vite), Tailwind CSS (Styling Engine)
* **Backend:** Node.js, Express.js
* **API:** Open-Meteo API
* **Code Editor:** Visual Studio Code

## 📱 How It Works

1. User signs up or logs into the system.
2. Admin/Organizers create and manage events.
3. Users browse, search, and filter available events.
4. Users register/book events.
5. Dashboard displays event insights and statistics.
6. Feedback and analytics help improve future events.

## 🎯 Objective

The main objective of **EventSync** is to provide a centralized platform that simplifies event organization, improves attendee engagement, and enhances event management efficiency.
